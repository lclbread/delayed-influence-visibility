# Measuring and Calibrating Delayed Influence Visibility in Pop/Rock Networks

Released analysis code for:

**Measuring and Calibrating Delayed Influence Visibility in Pop/Rock Networks**  
Chenglu Luo and Kaitai Guo, Xidian University

Public repository: https://github.com/ChengluLuo/delayed-influence-visibility

## Release Contents

This repository release contains only:

- `scripts/`
- `README.md`
- `requirements.txt`

The COMAP source CSV files and generated output figures are not redistributed.

## Overview

This package provides reference scripts corresponding to the empirical analyses reported in the manuscript. The release is intentionally limited to scripts and environment notes; it does not include source CSV files, generated figures, or generated numerical outputs. The code treats the ICM 2021 music data as documented relational data: artist-to-artist influence edges are used to measure graph reachability, not to infer causal transmission.

The final path and mediation claims in the paper are supported mainly by `02_low_degree_visibility.py` and `03_path_repertoire_checks.py`. The `01_context_baselines.py` script supports descriptive context, matched edge/non-edge baselines, cross-genre summaries, and sensitivity checks.

## Data

To run the scripts locally, create a local `data/` folder inside the release directory, or set `MUSIC_DATA_DIR` to the folder containing the COMAP ICM 2021 Problem D music CSV files. These are source-data files from the COMAP-provided music package, not files generated by the scripts:

- `data_by_artist.csv`
- `influence_data.csv`
- `full_music_data.csv`

Public redistribution of the CSV files should follow the COMAP ICM 2021 source-data license and redistribution conditions. The release repository does not include these CSV files. If the CSV files are absent, the scripts are expected to stop with a file-not-found error rather than silently substituting data.

Verified input MD5 hashes for the local analysis copy:

## Environment

Verified environment:

- Python 3.9.13
- NumPy 1.26.4
- Matplotlib 3.5.2

Install dependencies:

```bash
python -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
```

On macOS/Linux, use `source .venv/bin/activate` instead of the Windows activation command.

## Optional Local Reproduction

Run all scripts:

```bash
python scripts/run_all.py
```

Or run individual scripts:

```bash
python scripts/00_descriptive_context.py
python scripts/01_context_baselines.py
python scripts/02_low_degree_visibility.py
python scripts/03_path_repertoire_checks.py
python scripts/04_generate_path_subgraph.py
python scripts/05_effect_size_and_structure.py
```

By default, any locally generated figures are written to `outputs/figures/`. Generated outputs are not part of the release package. To override the output location, set `MUSIC_OUTPUT_DIR`.

## Script Map

- `00_descriptive_context.py`: preprocessing counts; descriptor-correlation heatmap; listed-follower cohort figure; descriptive follower-growth figure; Pop/Rock song-level decade boxplots.
- `01_context_baselines.py`: matched Pop/Rock edge/non-edge cosine baseline; 11-feature sensitivity; acoustic-extremeness and Mahalanobis sensitivity summaries; cross-genre comparison; Pop/Rock downsampling; dotplot and cross-genre figures.
- `02_low_degree_visibility.py`: early low-degree path-origin population; endpoint reachability by decade; endpoint concentration and overlap; temporal degree-preserving rewiring check.
- `03_path_repertoire_checks.py`: low-degree-to-late path enumeration; matched pseudo-path acoustic continuity; start-grouped path summaries; compact threshold grid; hub mediation; reachable-endpoint song-level descriptor profile.
- `04_generate_path_subgraph.py`: illustrative observed Pop/Rock path subgraph used as the manuscript graph visualization.
- `05_effect_size_and_structure.py`: standardized effect sizes; PCA/effective-dimension diagnostics; decade-standardized distance check; Pop/Rock network-structure, k-core, bow-tie, and hub-removal diagnostics; song-level cosine benchmark; compact cross-genre path diagnostic.

All random sampling uses seed `2026`. The matched edge/non-edge section uses 800 clustered bootstrap resamples and 3,000 sign-permutation draws. The path-level pseudo-path section uses 3,000 sign-permutation draws. The temporal rewiring check uses 1,000 rewired graphs by default. Set `MUSIC_REWIRE_ITERATIONS` to a smaller value for a quick smoke test or to a larger value for a more intensive Monte Carlo run. Set `MUSIC_REWIRE_ATTEMPTS_PER_EDGE` to change the number of swap attempts per edge within each rewired graph.

## Notes

Console output contains the numerical values used for the manuscript tables and prose summaries. Some generated figures are contextual diagnostics rather than final manuscript figures. Minor anti-aliasing and font-rendering differences can occur across Matplotlib backends.
